# Fake News Classification Using Machine Learning

This project builds a machine learning model to classify news articles as real or fake.

## Technologies
Python
Scikit-learn
Pandas
TF-IDF
Multinomial Naive Bayes

## Dataset
Kaggle Fake News Dataset

## Model
TF-IDF Vectorization + Multinomial Naive Bayes

## Accuracy
87%
